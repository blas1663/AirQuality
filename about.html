Tools

Leaflet is an open-source JavaScript library. This is what we used to create the map. We used Leaflet to change the location with the input boxes or with the moveend feature. We also used Leaflet to get the distance between the center of the map and the edge which we used as a radius.

Open AQ Platform is a collection of millions of air quality measurements from 9,973 locations. We used Open AQ to find the different locations of the measurements to put the marker. We also found the different measurements to put in the table.

Vue.js is an open-source JavaScript framework. We used Vue.js to create a Vue element and used databinding to update the coordinates and location as the map is moved. Vue.js was used throughout the entire website because it was a little easier and more condensed than using just regular JavaScript.

Location IQ is a collection of coordinates and their location names. The first part of the display name is the location of the coordinates. We can use Location IQ to find the location name from the coordinates or find the coordinates from the location name.
